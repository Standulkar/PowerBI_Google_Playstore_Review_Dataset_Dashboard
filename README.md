# PowerBI_Google_Playstore_Review_Dataset_Dashboard

Power BI Dashboard to visualize insights of Google Play store and user reviews dataset

Project Overview - This project is a Power BI dashboard built on Google Play Store apps and user reviews data. The goal is to analyze app performance, popularity, user satisfaction, and sentiment trends to help stakeholders understand what drives successful apps and where improvements are needed. The dashboard is divided into two focused pages: 1.App Performance Overview, 2.User Reviews & Sentiment Insights.
